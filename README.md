# BuscaVagaApp

O Aplicativo BuscaVaga é uma solução desenvolvida para a busca de estacionamentos privados em sua região, 
onde esses estacionamentos são retornados de um WebService.

### Linguagem utilizada
- Android SDK
- JAVA
